## Descripcion del sistema

Bragado es una aplicacion JavaFX para la gestion integral de un hotel. El sistema permite administrar habitaciones, huespedes, reservas, estadias, servicios adicionales, pagos simulados y reportes basicos de ocupacion.

La aplicacion contempla distintos tipos de usuarios:

- Administrador: gestiona habitaciones, huespedes, reservas, estadias, reportes y pagos.
- Recepcionista: gestiona reservas, check-in, check-out, servicios y pagos.
- Personal administrativo: consulta reservas, reportes y pagos.
- Huesped/cliente: consulta disponibilidad, crea reservas y administra sus propias reservas.

La interfaz grafica fue desarrollada con JavaFX puro y presenta una estetica moderna orientada a un software profesional para hoteles.

## Funcionalidades principales

- Gestion de habitaciones simples, dobles y suites.
- Control de estados de habitaciones: disponible, reservada, ocupada, limpieza y fuera de servicio.
- Registro y consulta de huespedes.
- Creacion, confirmacion, modificacion y cancelacion de reservas.
- Portal de cliente para consultar disponibilidad, reservar y ver reservas propias.
- Gestion de estadias mediante check-in y check-out.
- Incorporacion dinamica de servicios adicionales: desayuno, spa y estacionamiento.
- Calculo automatico del costo total de una estadia.
- Aplicacion de promociones y descuentos.
- Registro de pagos simulados y generacion de comprobantes.
- Notificaciones simuladas por email y SMS.
- Reportes basicos de ocupacion.
- Control de acceso por roles internos.

## Formas de uso

El sistema puede utilizarse de dos formas:

- Interfaz grafica JavaFX: version visual y principal del sistema.
- Terminal: version por consola desde la clase `Main`, con acceso a las mismas operaciones centrales: login interno, portal cliente, gestion de habitaciones, huespedes, reservas, check-in, check-out, servicios adicionales, reportes y pagos.
